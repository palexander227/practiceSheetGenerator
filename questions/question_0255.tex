


 Suppose $4x^{2}+1<17.$  What is the set of all $x$ that satisfies this inequality?

		\item   $x>4$

		\item  $-4<x<4$%
		\item  None of the above
	\end{enumerate}
\else
\fi

		\addtocounter{enumii}{1}

		\item  None of the above
	\end{enumerate}

\fi
