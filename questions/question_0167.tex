




 For some number $M$, a polynomial $f(x)$ satisfies $f(x)\leq M$ for every $x$.  Which of the following can be that polynomial?


\ifsat

		\item  $x^{2}-9$
		\item  $x^{2}$
		\item  $-x^{2}$%
		\item  $1-x$
	\end{enumerate}
\else
\fi

		\item  $-x^{2}$%
		\addtocounter{enumii}{1}
		\item  $x+1$
		\item  $1-x$
	\end{enumerate}
